About 01
I enjoy the full arc — from designing an interface to wiring up the database to deploying the thing and watching people use it.

My projects range from AI-powered coding agents to deterministic finance tools, each built with the same principle: every decision should be traceable, every interface should be honest, and nothing should be more complex than it needs to be.

I work primarily in Python and JavaScript, with Flask, React, and Node.js in the stack. I care about privacy-first design, clean documentation, and shipping things that actually get used.
